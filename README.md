# 🐙 dantecalderon API

My GraphQL API

💡 It provides information about my:

* Github Status (Github API)
* Contributions in last year(Github API)
* Latest commit(Github API)
* Listening(Last.fm API)
* Goodreads - Reading Status

### 🚀 Install and Run

```
yarn
```

then 

```
yarn dev
```
